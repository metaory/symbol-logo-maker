<h1 align="center">Symbol Logo Maker</h1>

<p align="center">Create beautiful symbol logos with customizable backgrounds, effects, and export options.</p>

---

## Features

- **Character or Icon mode** - Use any character/emoji or pick from 200k+ icons
- **Background shapes** - Rectangle, circle, or squircle with adjustable radius
- **Gradient fills** - Solid or gradient fills for background and symbol
- **Effects** - Drop shadow and glow with full control
- **Transform** - Rotation, skew, and alignment options
- **Export** - SVG download/copy and PNG at multiple resolutions

## Development

```bash
bun install
bun dev
```

## Stack

- Vite + React
- Tailwind CSS v4
- Zustand-style state with Proxy
- Iconify Picker
